◆ SPACE HOLOGRAM · S1 / S2 · v0.8 (3슬롯 구조 전환)
S1 · CENTRAL FLOOR 3D SPACE UPDATED · 24 마일스톤 → 3슬롯
중앙 바닥 공간. 홈 진입 시 두 주먹에 현재 이펙트가 라이브 적용된 상태로 보이고, 주변에는 이펙트 진행 상황을 보여주는 3슬롯(이전·현재·다음)만 노출. 24 마일스톤 전체는 위클리 스트릭 클릭 화면에서 확인.
◆ v0.8 STRUCTURE
주먹은 실제 적용 (라이브 100%), 슬롯 2는 같은 이펙트의 정적 디스플레이 (활성 80%). 주먹 = "살아있는 이펙트", 슬롯 = "진열장의 이펙트" 차별화. 슬롯 1은 딤드(이미 획득), 슬롯 3은 자물쇠(다음 목표) — 라벨 없이 시각만으로 커뮤니케이션.
SLOT 1 · PREV
state: 딤드 (어두운 톤, dashed 테두리)
content: 이펙트명 + WK 숫자
example: PHOENIX / WK 32
role: 이미 획득한 이펙트 회상
SLOT 2 · CURRENT
state: 활성 80% (가장 큰 박스, 두꺼운 테두리)
content: 이펙트명 + WK 숫자
example: SUPERNOVA / WK 60
role: 현재 적용 중인 이펙트의 정적 디스플레이
SLOT 3 · NEXT
state: 자물쇠 아이콘 (Lucide lock, 우상단)
content: 이펙트명 + WK 숫자 (윤곽만)
example: STARBURST / WK 80
role: 다음 목표 이펙트 예고
TWO FISTS · LIVE
position: 슬롯 하단 (유저 시점 자기 손)
state: 라이브 100% (현재 이펙트 풀 적용)
layer: 슬롯과 독립 — FIST 교체해도 이펙트 유지
differentiation: 살아있는 이펙트 vs 슬롯의 정적 이펙트
S2 · OVERHEAD RANK BADGE NO CHANGE
상공 RANK 배지. v0.4와 동일. FIST 요소 없음 (별도 UI 존재).
